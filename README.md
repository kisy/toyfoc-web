# ToyFOC web controller

`toyfoc` web 控制端程序

发送 `MQTT` 指令到 `ToyFOC-Ctrl` 后转换为 `I2C` 指令，再发送到 `ToyFOC`

配合 `ToyFOC-Ctrl` 使用
